================================================================================
COMMON ERRORS AND QUICK FIXES
================================================================================

ERROR: "API server not responding"
FIX:   - Check PowerShell terminal
       - See "Running on http://127.0.0.1:5000"?
       - If not: python api.py
       - Then try uploading again

ERROR: "Failed to read PCAP"
FIX:   - Try different PCAP file
       - Test with: python test_pcap.py mixed_attacks.pcap
       - If that works, use that file
       - If not, create new: python create_realistic_attacks.py

ERROR: Connection error or timeout
FIX:   - Browser Console (F12) → look for details
       - Restart everything:
         * Close api.py (Ctrl+C)
         * Close browser tab
         * python api.py
         * Refresh browser (F5)
         * Try again

ERROR: Blank screen, no error
FIX:   - Try different browser (Chrome, Edge, Firefox)
       - Clear browser cache: Ctrl+Shift+Delete
       - Run diagnostic: python test_frontend_diagnostic.py
       - Check browser Console (F12)
